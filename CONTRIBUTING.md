## How to contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-change`)
3. Commit your changes
4. Push your branch
5. Open a Pull Request

## Reporting issues

Please include:
- .NET version
- repro steps / minimal code
- expected vs actual behavior
